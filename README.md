# Accord Employee Timesheet

Printable static React + TypeScript web application that reproduces an Employee Time Sheet layout optimized for A4 landscape output.

## Tech Stack

- React 18
- TypeScript
- Vite
- Plain CSS

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Print Settings

- Open the page in browser.
- Use print mode and select A4 landscape.
- Keep "Background graphics" enabled so leave color coding is preserved.
- The stylesheet already includes print rules for page sizing and color retention.
